# Matt's dotfiles
Credit to [Mathias](https://github.com/mathiasbynens/dotfiles), [Gary Bernhardt](https://github.com/garybernhardt/dotfiles), and [Ivan Reese](https://github.com/ivanreese) for doing all the work.

## Installation

```bash
mkdir ~/dotfiles && cd $_
git clone https://github.com/mattfik/dotfiles.git && cd dotfiles && source bootstrap.sh
```

To update, `cd` into `dotfiles` and then:

```bash
source bootstrap.sh
```
